Shop.js

One Library For All Your Ecommerce Needs

Shop.js is a collection of highly customizeable e-commerce UI elements designed to support any checkout flow for Hanzo.

Get on GithubDownload

Getting Started

Include shop.js in your javascript and initialize using your Hanzo key to begin using it. Integrating shop.js Elements is as simple as inserting custom tags into your existing html page.

Carts

Carts display what items your customers have added to their carts, show customers theri totals, and allow customers to input coupons. In this example you case use the custom coupon GETSHOPJS.

Checkout

Checkouts allow users to input credit card information to pay for their purchases.

Side Cart

Carts can be embeded in a side pane and opened by triggering a 'side-pane-open' even with the id of the side pane.

Checkout Modal

Shop.js includes a prebuilt responsive checkout modal for quickly dropping into your ecommerce site. Start taking payments immediately while building out the rest of your online store.

And More!

Over the upcoming months, Shop.js will start supporting product cards, store pages, user self-management and much, much, more!
